# Unity Workspace Notes

This repository already contains a Unity project at `unity/SelfPlaySumoArena`.

## Intended Responsibilities

- `SelfPlaySumoArena/`: actual Unity project used for gameplay and final demo

- `scripts_plan.md`: script responsibilities and implementation order
